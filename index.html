Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, I assembled a dataset by pulling weather data on over 500 cities from the OpenWeatherMap API.

After assembling the dataset, I used the pandas and matplotlib libraries in Python to evaluate and plot several different weather metrics and how they change in respect to latitude. The metrics evaluated in this project were max temperature (F), cloudiness (%), wind speed (mph), and humidity (%).

I built this website to help visualize the findings after initially analyzing the data. I used HTML, CSS and Bootstrap that I learned during the DU Data Analytics and Visualization course. There was a lot of googling and rewatching examples from class in the make of this website.
